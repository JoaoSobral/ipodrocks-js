# Welcome

The Welcome screen is the first thing you see when you open iPodRocks.

## What it does

The Welcome panel gives you a quick overview of the app and links to the main areas: Library, Devices, Playlists, and Sync. It explains that iPodRocks works with any Rockbox or mountable device and guides you to use the sidebar for navigation.

## How it works

The Welcome screen is a static landing page. It does not load data from the database or devices. The "Get started" card lists the main panels and what they are for.

## How to work with it

1. Read the overview to understand the workflow: add a library folder, add a device, optionally create playlists, then sync.
2. Use the sidebar to jump to **Library**, **Devices**, **Playlists**, or **Sync**.
3. After your first setup, you can skip Welcome and go directly to **Dashboard** for an at-a-glance view of your library and devices.
