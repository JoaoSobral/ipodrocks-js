# Settings — Overview

Settings is a modal opened from the gear icon (top right). It has two sections: **OpenRouter API** and **Harmonic Analysis**.

## What it does

- **OpenRouter API** — API key and model for Savant playlists and the Music Assistant. Test connection before saving.
- **Harmonic Analysis** — Key/BPM extraction on scan, Essentia analysis, backfill percentage. Affects Library and Savant.

## How to work with it

1. Click the **gear icon** in the top-right of the app to open Settings.
2. Configure **OpenRouter** if you use Savant or the Assistant.
3. Configure **Harmonic Analysis** if you want key-aware mixing and Savant harmonic ordering.
4. Click **Save** to apply. **Cancel** discards changes.
